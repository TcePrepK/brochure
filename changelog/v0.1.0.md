# v0.1.0

## Features

- 2026-04-20: Shortcut `o` opens selected article link in system browser
- 2026-04-20: Shortcut `g` jumps to top of feed list
- 2026-04-20: Shortcut `C` collapses all categories in sidebar
- 2026-04-25: Pre-fetch on hover — silently fetch article content when navigating ArticleList (without waiting for Enter)
- 2026-04-21: Article age display — article detail shows time since publication in color
- 2026-04-21: Scrollbar — visual scroll indicator in all scrollable panels

- 2026-04-20: Feed title scroll animation starts faster (3-tick delay instead of 8)
- 2026-04-25: ArticleDetail keeps the three-panel layout instead of expanding to full width
- 2026-04-25: Star icon replaced — circle color encodes saved state: yellow=saved, blue=unread, dim=read
